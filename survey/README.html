How to use

This is a static demo UI. No data is stored. Refreshing resets everything.

Publish to GitHub Pages

  1. Create a new GitHub repository
  2. Upload all files (including index.html and the assets folder) to the repo root
  3. Go to Settings → Pages
  4. Choose Deploy from a branch
  5. Select main and folder / (root)
  6. Save, then open the URL that GitHub provides
