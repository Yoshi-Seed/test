# 卵巣がん患者向けWebインタビュースクリーニングシステム

## 概要
卵巣がん体験者を対象としたWebインタビューの一次スクリーニング用Webアプリケーションです。

## 特徴

### 🎯 主要機能
- **7つのYes/No質問**：段階的に表示される質問形式
- **自動スクリーニング**：1つでも「No」回答があった場合、即座にスクリーンアウト
- **申込フォーム**：すべての質問に「Yes」と回答した方向けの申込画面
- **メール連携**：申込情報を自動的にメールアプリで送信

### 🎨 デザイン特徴
- **医療調査に適した落ち着いた配色**：ブルー系の配色で安心感を提供
- **レスポンシブデザイン**：スマートフォン、タブレット、PCすべてに対応
- **配慮のあるUI/UX**：「審査されている」印象を与えない丁寧な表現
- **プログレスバー**：回答の進捗を視覚的に表示
- **スムーズなアニメーション**：画面遷移を自然に演出

## 質問内容

### Q1: 調査主体の確認
本調査は、製薬会社やマスコミによるものではありません。この点をご理解いただけますか？

### Q2: 機密管理の確認
調査内容は機密情報として適切に管理されます。この点をご理解いただけますか？

### Q3: 診断歴の確認
卵巣がん（タイプは問いません）と診断を受けたことがありますか？

### Q4: 診断時期の確認
卵巣がんと診断されたのは、10年以内（2015年以降）ですか？

### Q5: 治療状況の確認
現在、医療機関に通院し、薬による治療を受けていますか？

### Q6: 参加意思の確認
2025年3月3日～3月13日の間に、60分程度のWebインタビュー（カメラOFF）にご参加いただくご意思はありますか？
※PCまたはiPad等の端末が必要です（スマートフォンのみ不可）

### Q7: 個人情報共有の同意
今回のインタビュー実施に必要な事務連絡を目的として、お名前・メールアドレスを、調査実施会社である株式会社シード・プランニングに共有することに同意いただけますか？

## 画面フロー

1. **質問画面**：1問ずつ表示、Yes/Noで回答
2. **分岐**：
   - 1つでも「No」→ スクリーンアウト画面へ
   - すべて「Yes」→ 申込フォーム画面へ
3. **スクリーンアウト画面**：丁寧なお詫びメッセージ
4. **申込フォーム画面**：個人情報入力→メール送信

## 使用方法

### 基本的な使用
1. Webブラウザで `index.html` を開く
2. 質問に順番に回答する
3. すべて「はい」の場合：申込フォームが表示される
4. 1つでも「いいえ」の場合：お詫び画面が表示される

### メール設定のカスタマイズ
`script.js` の以下の部分を編集してください：

```javascript
// Line 156付近
const recipientEmail = 'interview@example.com'; // 実際のメールアドレスに変更
```

### デプロイ方法

#### 静的ホスティングサービスの場合
- **GitHub Pages**：リポジトリにプッシュ後、Settings > Pages で有効化
- **Netlify**：ディレクトリをドラッグ&ドロップ
- **Vercel**：GitHubリポジトリと連携
- **Cloudflare Pages**：GitHubリポジトリまたはディレクトリアップロード

#### サーバーの場合
単純なHTTP静的ファイルサーバーで動作します：

```bash
# Python 3
cd screening
python3 -m http.server 8000

# Node.js (http-server)
npx http-server screening -p 8000
```

その後、ブラウザで `http://localhost:8000` にアクセス

## カスタマイズガイド

### 色の変更
`styles.css` の `:root` セクションで色を変更できます：

```css
:root {
    --primary-color: #4a6fa5;      /* メインカラー */
    --success-color: #5a9e6f;      /* 成功時の色 */
    --text-primary: #2c3e50;       /* メインテキスト色 */
    --background: #f5f7fa;         /* 背景色 */
}
```

### 質問内容の変更
`script.js` の `questions` 配列を編集：

```javascript
const questions = [
    {
        id: 1,
        text: "質問内容をここに入力"
    },
    // ...
];
```
