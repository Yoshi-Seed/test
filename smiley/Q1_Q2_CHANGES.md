# Q1・Q2 文言変更報告

## 📝 変更概要

Q1とQ2の文言をより分かりやすく、患者さんにとって理解しやすい表現に変更しました。

---

## 🔄 Q1の変更

### Before（変更前）
```
本調査は、製薬会社やマスコミによるものではありません。
この点をご理解いただけますか？
```

❌ **問題点**:
- 否定文から始まり、何を聞かれているのか分かりにくい
- 「理解できるか」という曖昧な質問
- 製薬会社やマスコミで働いている人が「いいえ」を選ぶべきか不明確

---

### After（変更後）
```
このインタビューには特定のお仕事をされている方が除外されます。ご理解ください。

あなたは、製薬会社またはマスコミで勤務されていますか？

※勤務されている場合は「いいえ」を選択してください
```

✅ **改善点**:
1. **前提の説明**: 「特定のお仕事をされている方が除外される」と明示
2. **直接的な質問**: 「あなたは〜勤務されていますか？」と明確に
3. **強調表示**: `<strong>`タグで質問を強調
4. **具体的な指示**: 「勤務されている場合は『いいえ』を選択」と明記
5. **スタイリング**: 注釈は小さめの文字とグレー色で視覚的に区別

---

## 🔄 Q2の変更

### Before（変更前）
```
調査内容は機密情報として適切に管理されます。
この点をご理解いただけますか？
```

❌ **問題点**:
- 一方的な情報提供のみ（患者側の義務が不明）
- 「調査内容」という抽象的な表現
- 同意書が別途必要という重要情報が欠落

---

### After（変更後）
```
インタビューで見たり聞いたりする情報を他に共有しない同意書に別途ご同意が必要です。

同時に、あなたの情報も機密情報として適切に管理されます。

ご理解いただけますか？
```

✅ **改善点**:
1. **双方向の義務を明示**:
   - 参加者側: インタビュー内容を他に共有しない
   - 主催者側: 参加者の情報を適切に管理
   
2. **具体的な表現**:
   - 「調査内容」→「インタビューで見たり聞いたりする情報」
   - より具体的でイメージしやすい
   
3. **重要情報の追加**:
   - 「別途ご同意が必要です」を明記
   - 守秘義務契約があることを事前に認識できる
   
4. **段落構成**:
   - `<br><br>`で段落を分け、読みやすく
   - 情報の優先順位を視覚的に整理

## 💡 変更のポイント

### 1. **質問形式の統一**
- 曖昧な「理解できるか」ではなく
- 明確な「該当するか/同意するか」の質問に

### 2. **視覚的な強調**
```javascript
// 主要な質問部分を<strong>で強調
text: "あなたは、<strong>製薬会社またはマスコミで勤務されていますか？</strong>"

// 注釈をスタイル付きのsmallタグで
text: "<small style='display:block; margin-top:12px; color:#8b7970; line-height:1.7;'>
       ※勤務されている場合は「いいえ」を選択してください</small>"
```

### 3. **段落構成の改善**
```javascript
// 情報を論理的に区切る
text: "前提説明<br><br>主要質問<br><br>補足説明"
```

- ✅ 具体的な例を使用（イメージしやすさ）
